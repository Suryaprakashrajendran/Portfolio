# Portfolio Starter (Vite + React + Tailwind)

## Run Locally
```bash
npm install
npm run dev
```

Open the URL shown in your terminal (usually http://localhost:5173).

## Build
```bash
npm run build
npm run preview
```

## Edit your details
- Open `src/App.jsx` and update the `profile`, `projects`, and `experience` objects.
- Replace `profile.links.resume` with a real PDF link.
